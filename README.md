# Retro-UI

```txt
project/
├── main.py                # Entry point of the application
├── ui/
│   ├── __init__.py        # Makes this a package
│   ├── window.py          # Main window and rendering logic
│   ├── widgets.py         # Classes for UI elements like buttons, sliders, etc.
│   ├── styles.py          # Styling constants (e.g., colors, fonts, sizes)
├── app/
│   ├── __init__.py        # Makes this a package
│   ├── event_handler.py   # Handles events (keyboard, mouse, etc.)
│   ├── main_loop.py       # Contains the main loop logic
├── data/
│   ├── assets/            # Static assets (fonts, images, sounds)
│   ├── config.py          # Configuration settings
│   ├── data_loader.py     # Loads external data or settings
└── utils/
    ├── __init__.py        # Makes this a package
    ├── helpers.py         # Utility functions
```
